# JavaScript Calculator Practice

## Overview
This project demonstrates basic arithmetic operations and DOM manipulation using JavaScript. The user can click buttons to perform addition, subtraction, division, or multiplication on two numbers, and the result is displayed dynamically in the browser.

## What I Learned
- How to display JavaScript variables in HTML using `.textContent`
- How to create multiple functions for different operations
- How to update the DOM based on user interaction
- Basic calculator logic using functions
